# Responsive Watch Store Website

A simple responsive watch store website developed using HTML, CSS, JavaScript, and jQuery to familiarize basic frontend web development concepts.

## Features

- Responsive product catalog layout
- Product display with image, name, code, and price
- Buy Now functionality
- Enquiry form
- Product detail display
- Confirmation page
- About company section

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery

## Purpose

This project was developed as a beginner-level mini project to practice and understand fundamental concepts of frontend web development.
